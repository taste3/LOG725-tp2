## Le jeu à été testé avec
- Python 3.13.7
- pip 25.2
- Windows

## Comment lancer le jeu modifié
1. Naviguer dans le dossier <strong>LOG725-TP2\myriapod-refactored</strong>
2. Installer les requis avec la commande ```pip install -r requirements.txt```
3. Lancer le jeu avec la commande ```python .\myriapod.py```

## Comment lancer le jeu original (sans les modifications)
1. Naviguer dans le dossier <strong>LOG725-TP2\myriapod-refactored</strong>
2. Installer les requis avec la commande ```pip install -r requirements.txt```
3. Naviguer dans le dossier <strong>LOG725-TP2\myriapod-original</strong>
4. Lancer le jeu avec la commande ```python .\myriapod.py```
